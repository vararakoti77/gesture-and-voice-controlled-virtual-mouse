# 🖐️ Quick Gesture Reference Card

## Essential Gestures (Master These First!)

| Gesture | Fingers | Action | Emoji |
|---------|---------|--------|-------|
| **Move Cursor** | Index only | Control mouse | 👆 |
| **Click** | Thumb + Index pinch | Left click | 👌 |
| **Scroll** | Index + Pinky | Scroll up/down | 🤙 |

## Quick Tips

✅ **Best for Clicking**: Use Pinch Click (Gesture #2)
✅ **Best for Moving**: Use Index Finger (Gesture #1)  
✅ **Best for Scrolling**: Use Index+Pinky (Gesture #7)

---

## Common Combinations

**Web Browsing**:
1. Move cursor (Index finger)
2. Click links (Pinch)
3. Scroll page (Index+Pinky)
4. Right click (3 fingers)

**Document Editing**:
1. Move cursor (Index finger)
2. Select text (Index+Middle apart)
3. Double click word (Thumb+Index+Middle)
4. Scroll document (Index+Pinky)

**Media Playback**:
1. Play/Pause (4 fingers)
2. Volume (Thumb+Pinky)
3. Screenshot (Open palm)

## How to Start

1. Run: `python AiVirtualMouseProject.py`
2. Position hand in purple rectangle
3. Wait for green landmarks
4. Start with Gesture #1 (Move)
5. Try Gesture #2 (Pinch Click)
6. Practice other gestures

---

## Troubleshooting Quick Fixes

❌ **Not detecting hand?** → Better lighting
❌ **Gestures not working?** → Make clearer finger positions
❌ **Cursor jumpy?** → Move hand smoother
❌ **Camera not found?** → Close other apps using camera
